# 🏥 AI Medical Image Analyzer

An AI-powered web application that analyzes medical images and generates detailed observations using artificial intelligence.

This project is built using **Python** and **Flask** and integrates with **OpenAI vision models** to analyze uploaded medical images such as X-rays, CT scans, MRIs, or dermatology images.

The system processes the image and returns AI-generated insights to help understand potential patterns in the image.

⚠️ **Disclaimer:** This tool is for educational and research purposes only and should not be used as a replacement for professional medical diagnosis.

---

## 🚀 Features

- Upload medical images for analysis
- AI-powered medical image interpretation
- Detailed observations generated by AI
- Simple and interactive web interface
- Secure file upload handling
- Automatic image processing and analysis

---

## 🛠️ Tech Stack

- **Python**
- **Flask**
- **OpenAI Vision API**
- **HTML / CSS**
- **Base64 Image Encoding**

---


## 📸 How It Works

1. User uploads a medical image.
2. The image is securely saved and encoded into Base64 format.
3. The encoded image is sent to the OpenAI vision model.
4. The AI analyzes the image and generates observations.
5. The results are returned and displayed in the web interface.
